## ROUTES

<hr>

### BLOG

<hr>

#### GET ALL BLOGS
```cmd
    GET /api/v1/blog/
```

#### GET ONE BLOG
```cmd
    GET /api/v1/blog/<id>
```

#### CREATE BLOG
```cmd
    POST /api/v1/blog/
```

#### UPDATE BLOG
```cmd
    PUT /api/v1/blog/<id>
```

#### DELETE BLOG
```cmd
    DELETE /api/v1/blog/<id>
```
